Frequently Asked Questions

Do I need coding skills to use Guest Posts?
No! Guest Posts is designed for non-technical users. The setup process is straightforward, and the plugin provides clear step-by-step instructions.
How does keyword filtering work?
Each site in your network can set keywords. When a post is published, it checks if the post title, content, tags, or categories match any of the keywords. If there's a match (OR logic), the post is published. If no keywords are set, all posts are accepted.
Is Guest Posts compatible with Elementor?
Yes! Guest Posts has full Elementor support. Your Elementor-designed pages and posts will cross-post perfectly, preserving all styling and widgets.
What happens when I update a post?
When you update a published post, Guest Posts automatically syncs the changes to all connected sites where the post was previously published.
How secure is the API key system?
API keys are encrypted using WordPress's secure encryption methods. Your network connections are protected with industry-standard security practices.
Can I use Guest Posts with unlimited sites?
Yes! There are no limits on the number of sites you can connect. Add as many WordPress sites to your network as you need.
